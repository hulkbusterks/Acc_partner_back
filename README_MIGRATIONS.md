Alembic migrations
-------------------

Run migrations (PowerShell):

```powershell
$env:PYTHONPATH = "$PWD"
.\.venv\Scripts\python.exe -m alembic upgrade head
```

Create an autogenerated revision after changing `server/models.py`:

```powershell
$env:PYTHONPATH = "$PWD"
.\.venv\Scripts\python.exe -m alembic revision --autogenerate -m "describe change"
.\.venv\Scripts\python.exe -m alembic upgrade head
```

If `PYTHONPATH` is not set, Alembic may not import your application; the snippet above ensures the project root is discoverable.
